Testando Menus

Testando menus com exemplos práticos

Qualquer elemento HTML que descreva texto (como p, h1, h2, ul, ol, entre outros) pode ser estilizado para criar menus. No entanto, o elemento mais adequado, flexível e fácil de estilizar, do ponto de vista semântico, são as listas. Seja uma lista de definição (dl), uma lista ordenada (ol), ou uma lista não ordenada (ul), esses elementos oferecem os melhores recursos para a construção de menus.
